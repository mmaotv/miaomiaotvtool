## 新增UA伪装+首页自定义 (15:14)

### CONFIG.txt 新增配置项
```
[直播按钮]    LIVE_NAME / LIVE_URL
[点播按钮]    VOD_NAME / VOD_URL
[工具按钮]    TOOLS_NAME / TOOLS_URL
[扫码按钮]    QR_NAME（功能固定）
[文件按钮]    FILE_NAME（功能固定）
[U盘按钮]    USB_NAME（功能固定）
[浏览器UA]   USER_AGENT（chrome/edge/safari/mobile/desktop/custom）
```

### UA类型
- chrome: 电脑Chrome | edge: Edge | safari: Mac Safari
- mobile: 手机浏览器 | desktop: 电脑通用 | custom: 自定义
